From Coding Challenges to Watercress AI

11 October '24

It's been two weeks since my last blog post, and a lot has happened in that time. The time has flown by—the watercress is starting to grow, and I'm flooded with new information.

I've learned how to set up my own REST API in Node.js, use Postman to test the API, and MySQL to store the data. I completed the assignment for the REST API chapter, where I had to create a REST API that could store, retrieve, update, and delete data. Since I had never worked with REST APIs before, it was quite a challenge, but I managed to get it working, and I'm proud of the result.

I also had my first assignment for C++ programming, where I needed to write a program that could read a log file and extract information from it. The first part of the assignment went smoothly, but the second part was trickier—I had to display the top 3 most logged items in the file. It wasn't as easy as I thought, but with some help and a lot of trial and error, I eventually got it working.

My first continuous evaluation in AI programming with Python went well. The initial assignments were pretty straightforward, but the last three were a bit more complex. I had to write a program that could read in a table from a specific website, extract the data, and display it in a graph.

The most exciting part of the past two weeks has definitely been the Watercress AI project. The watercress is growing well, and as it develops, I'm taking daily photos to document its progress. Some plants are growing faster than others, while a few aren't growing at all, giving us a nice variety of growth stages to study. I'm labeling the photos using Labelbox, a tool that makes it easy to label images for machine learning.

I brought the Raspberry Pi home to learn how to use it and get more comfortable with it. I successfully made an SSH connection to it, which means I can work on it from my laptop. Getting that to work felt like a huge accomplishment! I also created a simple program that takes a picture using the button on the Sense HAT and displays the number of pictures taken on the LED matrix. It wasn't as easy as I thought it would be— I had to look up a lot of things, but that's how you learn, right?

The project leader has asked us to create a dashboard where we can see the pictures we've taken, the predicted age of the watercress, and the actual age of the watercress using a Streamlit app. I've never worked with Streamlit before, so I'm excited to dive into it and learn how to use it. There's a lot of work ahead, but I'm feeling optimistic and ready for the challenge!

Overall, it's been a busy and rewarding couple of weeks, and I'm excited to see where this project will lead. I'm looking forward to learning more and sharing my progress with all of you!

Rise of the watercress

27 September '24

The first week of school has officially wrapped up, and what a week it has been! With a few new teachers and bunch of interesting new courses, things are off to an exiting start. Some of the classes felt like a review of things i had seen before, like when to use let, const, or var in JavaScript, or the basics of C++ programming. But there were also a few things I hand't encountered before, which made it all the more exciting.

For example, in my IoT Devices course, I learned about a much easier way to assign a function to a specific IO pin. We used a program called STM32CubeMX, where you can select the microcontroller you're working with, configure the pins, and then generate the code to use in Keil µVision. The whole process was so simple and intuitive. Now, I'm wandering if it will stay that way?

Since I'm in my second year, I have a more elective courses than last year. One of them is AI programming in Python, where I'll learn how to build and train an AI model. To develop a good AI model, I'll need to know how to collect and interpret data effectively. I'm really excited to dive into AI and everything that comes with it. There's so much potential in this field, and I can't wait to learn more.

Now, for the moment you've all been waiting for: What project did I get?

Drummroll please...

I'm on the Watercress AI project team! This week, we had our first meeting with the project leader, who gave us a lot of insight into what we'll be working on. He's a researcher at the school and has extensive knowledge in AI, machine leaning, and deep learning.

The goal of the project is to create a standalone device that can predict the growth stage of watercress plants based on a picture. We'll be using a Raspberry Pi with a camera and a display to show the predicted age of the watercress.

The plan

To get accurate date, we'll plant several batches of watercress at home. Every day, we'll take pictures from different angles, in various lighting conditions, and with different cameras. Then, we'll label those photos and divide them into three data sets: a training set to teach the AI model, a validations set to fine-tune it, and a test set to evaluate how well the model performs.

Every Thursday, we have a full day dedicated to the project. This week's goal was to brainstorm and come up with a solid action plan for the next few weeks. There's a lot of work ahead, but I'm feeling optimistic! We've already tried out the Raspberry Pi and the camera, so we're comfortable with the technology. I can't wait to see how this project unfolds.

Stay tuned for more updates on the Watercress AI project!
