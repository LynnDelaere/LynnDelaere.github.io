Rise of the watercress

27 September '24

The first week of school has officially wrapped up, and what a week it has been! With a few new teachers and bunch of interesting new courses, things are off to an exiting start. Some of the classes felt like a review of things i had seen before, like when to use let, const, or var in JavaScript, or the basics of C++ programming But there were also a few things I hand't encountered before, which made it all the more exciting.

For example, in my IoT Devices course, I learned about a much easier way to assign a function to a specific IO pin. We used a program called STM32CubeMX, where you can select the microcontroller you're working with, configure the pins, and then generate the code to use in Keil µVision. The whole process was so simple and intuitive. Now, I'm wandering if it will stay that way?

Since I'm in my second year, I have a more elective courses than last year. One of them is AI programming in Python, where I'll learn how to build and train an AI model. To develop a good AI model, I'll need to know how to collect and interpret data effectively. I'm really excited to dive into AI and everything that comes with it. There's so much potential in this field, and I can't wait to learn more.

Now, for the moment you've all been waiting for: What project did I get?

Drummroll please...

I'm on the Watercress AI project team! This week, we had our first meeting with the project leader, who gave us a lot of insight into what we'll be working on. He's a researcher at the school and has extensive knowledge in AI, machine leaning, and deep learning.

The goal of the project is to create a standalone device that can predict the growth stage of watercress plants based on a picture. We'll be using a Rasberry Pi with a camera and a display to show the predicted age of the watercress.

The plan

To get accurate date, we'll plant several batches of watercress at home. Every day, we'll take pictures from different angles, in various lighting conditions, and with different cameras. Then, we'll label those photos and divide them into three data sets: a training set to teach the AI model, a validations set to fine-tune it, and a test set to evaluate how well the model performs.

Every Thursday, we have a full day dedicated to the project. This week's goal was to brainstorm and come up with a solid action plan for the next few weeks. There's a lot of work ahead, but I'm feeling optimistic! We've already tried out the Rasberry Pi and the camera, so we're comfortable with the technology. I can't wait to see how this project unfolds.

Stay tuned for more updates on the Watercress AI project!
